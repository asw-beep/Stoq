# AI Financial Intelligence Platform

## Vision

Build a production-grade financial intelligence platform that combines:

1. Stock Forecasting
2. Financial News Sentiment Analysis
3. Portfolio Analytics
4. Financial Dashboards
5. Cloud Deployment

The project serves as a demonstration of:

- Machine Learning
- Time-Series Forecasting
- NLP
- Backend Engineering
- Frontend Development
- DevOps
- Cloud Engineering

---

## Target Users

Retail investors
Students
Researchers

---

## Core Features

### F1 Stock Forecasting

Input:
- Stock Symbol

Output:
- Historical Prices
- 1-Day Forecast
- 7-Day Forecast
- 30-Day Forecast

Models:
- Prophet
- XGBoost

---

### F2 News Sentiment Analysis

Input:
- Company Symbol

Output:
- Bullish Score
- Bearish Score
- Neutral Score

Model:
- FinBERT

---

### F3 Portfolio Analytics

Users can:

- Create portfolios
- Add holdings
- View performance

Metrics:

- Return
- Volatility
- Sharpe Ratio
- Drawdown

---

### F4 Dashboard

Market Overview

Forecast Dashboard

Portfolio Dashboard

Sentiment Dashboard

---

## Non Goals

No trading execution.

No brokerage integration.

No real-time high-frequency trading.
